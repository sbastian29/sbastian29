Soy **Ingeniero de Datos** cursando un Máster en Data Engineering, con foco en construir **pipelines de datos completos** y aplicar **Machine Learning** a problemas reales de negocio.

- 🔭 Actualmente trabajo con infraestructura de datos: **Airflow, dbt, OpenMetadata** y modelado de pipelines.
- 🎓 Desarrollando mi **TFM sobre análisis de churn**: arquitectura Big Data con Kafka → Spark → PostgreSQL y modelado predictivo.
